Disseny gràfic

Disseny gràfic

Ens introduirem en el mon del disseny vectorial amb Inkscape que ens permetrà utilitzar formes geomètriques i vectors per a crear logotips, cartells i senyals tan complexes com vulguem, podent imprimir-les a qualsevol tamany sense perdre qualitar.

Amb Gimp explorarem el mon de la fotografia digital, retocs, filtres i color. També treballarem amb eines que ens permetran fer muntatges originals.

Modelatge 3D

Ens introduirem en el mon de les 3 dimensions amb tinkercad, programa de modelatge online que ens permetrà crear els nostres objectes, així com utilitzar objectes creats per altres i publicats a repositoris en línia.

Els alumnes amb major nivell provaran Blender, una eina de modelatge i animació en 3 dimensions, més complexe i amb més opcions.

Disseny web

Cream programes utilitzant distints llenguatges de programació

  • Aprenem a dissenyar i proramar pàgines web amb els llenguatges HTML, CSS i Javascript, per tal de fer llocs webs dinàmics i adaptables als diferents dispositius
  • Els grups més avançats, des del curs 22-23, proven frameworks de frontend com a bootstrap per als seus projectes web.
  • Github ens permet guardar al núvol els nostres projectes, així com a col·laborar amb altres programadors i dur un control de les versions dels nostres projectes
  • Aprendrem a utilitzar entorns IDE, com ara Visual Studio Code per a programar de forma més eficient
  • Com a darrer pas, aprenem a publicar la nostra web a Internet per tal de que es pugui veure des de tot el món.
